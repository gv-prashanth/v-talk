Getting Started

I made sure that using V-Talk is as easy as double clicking an icon (in case ur wondering what it means, then this section is definitely for u). Let me assure that ur just 3 steps away from chatting with ur friend online. just follow the following steps to get it started.

  1. Go to the downloads section and click the download button to download the latest version of V-talk available.
  2. In the pop up menu, select "Save As" option and then point to ur destination directory, click save.
  3. When download completes, double click vtalk.exe in the downloaded destination.

Congratulations !!! ur now a member of our V-Talk user community. Just type ur required chat and then press enter to send. U can change ur Nick (Default: User1) by typing in the user nick text field.
