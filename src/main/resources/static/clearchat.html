About V-Talk

The following are some the salient features of this software:

  • Simple and easy to use.
  • No need to install and all. Just download the software and double click to start chatting.
  • No need to set ur proxy settings. The program will search automatically for network connections.
  • Small in size. Download - Chat - Delete. Its as simple as that !!!

I would also like to give u limitation of this software as of now. I will try to get a better version with fewer restriction asap.

Few limitations:

  • There may be a few seconds (2-3 sec) lag while chatting.
  • Sometimes if ur typing chat quickly 1 after other, some of the lines may be missing. I suggest that u type a longer chat and then press enter instead of short chats.
  • Further, there may be a little lag in the user interface. But this wont crash ur computer on nything of that sort. So plzz don't worry if u notice such lags.
  • Note that as of now, there is only ONE chat room. If u want to chat privately, then this is not the software for u.
